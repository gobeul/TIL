### 튜플 (시퀀스 - 불변형)

여러개의 값을 순서가 있는 구조로 저장하고 싶을 때 사용
리스트와 차이는 값 변경이 불가능 함 (불변 자료형)

`a = ()  or  b = tuple() # 튜플 생성`

수정은 불가능(불변형) 하지만 인덱스로 값에 대한 접근은 가능(시퀀스)


주의!

- 단일 항목의 경우 뒤에 쉼표(,) 를 붙여야 함 (안 붙이면 튜플로 만들어 지지 않는다.)⇒ my_tuple = (1,)
  ```python
  a = (1)
  print(a, type(a))
  >> 1 <class 'int'>

  a = ("가")
  print(a, type(a))
  >> 가 <class 'str'>

  a = ("가",)
  print(a, type(a))
  >> ('가',) <class 'tuple'>
  ```
- 복수 항목의 경우 마지막에 쉼표가 필요는 없지만 붙이는 것을 권장. 습관들이는게 좋음.

튜플은 직접적으로 사용되는 일은 잘 없지만, 파이썬 내부에서 다양하게 활용된다.
